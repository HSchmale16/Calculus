# What is the Mandelbrot set?
* A set of complex numbers for which a repeating pattern does **NOT**
  approach infinity.
* It is described by $z_{n+1} = z_{n}^2+c$ remains bounded.
    * That means that $c$ is a member of the mandelbrot set
